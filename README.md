Веб-приложение для поиска авиабилетов с интерфейсом

## Реализовано

- Поиск авиабилетов по направлениям
- Выбор города отправления и прибытия
- Фильтрация результатов
- Сортировка билетов
- Работа с API

## Технологии

Frontend:
- React
- React Router
- Axios
- JavaScript
- CSS/SCSS

Backend:
- Node.js
- Express
- API integration
